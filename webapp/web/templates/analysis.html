Interesting Data Analysis

Recommended Game Genres
Hover over an edge segment to see all recommended genres for a specific game genre. Hover over a connection to see the percentage of all recommendations in each direction (chosen genre to recommended genre).

Using the IGN commenter dataset, we can look at which game genres are most insular. That is: when showing recommendations for a particular game, how many are from the same genre?

Because the classifier is built based on user interest, this is an interesting question for game developers: if I hook users with a particular game, should I make my next game in the same genre? Conventional wisdom would say yes, but exploring this with empirical data could be illuminating.

A chord plot illustrating the connections between input game genre and recommended ones, is shown on the left.

The most insular category, by a large margin, is Wrestling (pink segment near the top), where 14% of all recommendations for wrestling games are also in the wrestling genre. As wrestling games are only 0.4% of the entire set, this is a 35x increase as compared to random. There appears to be a strong niche group which prefers to play most wrestling games that are released.

The next most insular category is Party (small blue segment near the bottom), which may be due to more casual gamers. The fraction of party games recommended from other party games is 9x higher than the overall percentage of party games in the dataset.

Most large game categories have insular factors of roughly 2-4x, indicating that people who enjoy a certain game category prefer other games in the exact same category.

Validation
Prediction success for users from Gamespot using the RespawnInto recommendations vs best-in-genre vs random chance.

To validate the IGN commenter algorithm, recommendations were compared to an outside source. Similar to the method used to obtain the IGN data, commenter names were scraped from over 300 game review pages on gamespot.com.

For the more than 1400 gamespot commenters who commented on 2+ games, one of the games was fed into the RespawnInto algorithm. The user was classified as "correctly predicted" if the other game appeared in some top number of recommended games.

The successful prediction rate by the RespawnInto recommendation algorithm is compared to two other methods in the plot to the left:

  • Random: The requested number of games were chosen randomly from the full set of 855 that have IGN Comments.
  • Best-In-Genre: Only games in the same genre as the selected game were returned, and were ordered by IGN Score.

The final version of RespawnInto displays 12 recommendations in order to make the interface as simple as possible. In this scenario, the RespawnInto algorithm is 75% better than the best-in-genre selection, and 30 times better than random.
