Data Set

The data for this project was scraped from IGN.com, a video game review website. From IGN, the following information was obtained:

  • Basic game information, like Name, Platform, Developer, Genre and Release Date on nearly 6500 games.
  • Game ratings from the IGN reviewer, as well as the aggregate community score for 6000 games.
  • Full review text on nearly 900 games.
  • List of top-level commenters and comment text on 850 games.

Algorithm

Using this information, several recommendation systems can be implemented:

  • Jaccard Similarity: By scraping commenter usernames, individuals can be tracked between games. Using the amount of overlapping commenters, a Jaccard Similarity measure (cosine similarity) can be obtained between games. This similarity measure is used for the bulk of video games in the dataset.
     
  • TF-IDF: For games without commenter information, or those games with no overlapping commenters, the full review text of all games in the dataset is used. A Term Freqency-Inverse Document Frequency analysis of the selected game's review highlights other game reviews about similar topics.
